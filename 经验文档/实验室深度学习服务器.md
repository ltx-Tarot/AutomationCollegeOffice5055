### 🔄 3.2 多窗口管理：`tmux`

使用 `tmux` 命令进行终端分屏与多任务管理，避免断开 SSH 导致程序终止。

常用命令：

```bash
tmux new -s session_name    # 新建会话
tmux ls                     # 查看会话列表
tmux attach -t session_name # 恢复会话
tmux kill-session -t name   # 关闭会话
```
